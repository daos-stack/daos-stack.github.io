◆ co_post_reply

int(* co_post_reply(crt_rpc_t *rpc, void *arg)

Collective RPC post-reply callback. This is an optional callback. If specified, it will execute after reply is sent to parent (after co_aggregate executes).

Parameters
[in]rpcthe rpc structure of the parent
[in]argthe private pointer, valid only on collective RPC initiator (same as the priv pointer passed in for crt_corpc_req_create). Can be used to share data between co_aggregate and co_post_reply when executing those callbacks on the same node.

int(* co_pre_forward(crt_rpc_t *rpc, void *arg)

Collective RPC pre-forward callback. This is an optional callback. If specified, it will execute prior to corpc request being forwarded.

Parameters
[in]rpcthe rpc structure
[in]argthe private pointer, valid only on collective RPC initiator (same as the priv pointer passed in for crt_corpc_req_create).
Return values
Anyvalue other than DER_SUCCESS will cause CORPC to abort.
