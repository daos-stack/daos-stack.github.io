◆ to_check_availability

int(* to_check_availability(struct btr_instance *tins, struct btr_record *rec, uint32_t intent)

Optional: Check whether the given record is available to outside or not.

Parameters
tins[IN] Tree instance which contains the root umem offset and memory class etc.
rec[IN] Record to be checked.
intent[IN] The intent for why check the record.

return ALB_AVAILABLE_DIRTY The target is available but with some uncommitted modification or garbage, need cleanup. ALB_AVAILABLE_CLEAN The target is available, no pending modification. ALB_UNAVAILABLE The target is unavailable. -DER_INPROGRESS If the target record is in some uncommitted DTX, the caller needs to retry related operation some time later. Other negative values on error.

◆ to_hkey_cmp

int(* to_hkey_cmp(struct btr_instance *tins, struct btr_record *rec, void *hkey)

Optional: Comparison of hashed key.

Absent: Calls memcmp.

Parameters
tins[IN] Tree instance which contains the root umem offset and memory class etc.
rec[IN] Record to be compared with key.
key[IN] Key to be compared with key of rec.

return BTR_CMP_LT hkey of rec is smaller than hkey BTR_CMP_GT hkey of rec is larger than hkey BTR_CMP_EQ hkey of rec is equal to hkey BTR_CMP_ERR error in the hkey comparison return any other value will cause assertion, segfault or other undefined result.

int(* to_key_cmp(struct btr_instance *tins, struct btr_record *rec, d_iov_t *key)

Optional: Comparison of real key. It can be ignored if there is no hash for the key and key size is fixed. See btr_record for the details.

Absent: Skip the function and only check rec::rec_hkey for the search.

Parameters
tins[IN] Tree instance which contains the root umem offset and memory class etc.
rec[IN] Record to be compared with key.
key[IN] Key to be compared with key of rec.

return BTR_CMP_LT key of rec is smaller than key BTR_CMP_GT key of rec is larger than key BTR_CMP_EQ key of rec is equal to key BTR_CMP_ERR error in the key comparison return any other value will cause assertion, segfault or other undefined result.
