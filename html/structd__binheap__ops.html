bool(* hop_compare(struct d_binheap_node *a, struct d_binheap_node *b)

A binary predicate which is called during internal heap sorting, and used in order to determine the relevant ordering of two heap nodes.

Implementing this operation is mandatory.

Parameters
[in]aThe first heap node
[in]bThe second heap node
Returns
true if node a < node b, false if node a > node b.
See also
d_binheap_bubble() and d_biheap_sink()
