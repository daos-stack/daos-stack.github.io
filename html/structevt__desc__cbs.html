Detailed Description

Callbacks and parameters for evtree descriptor

NB:

  • evtree is a standalone algorithm, it should not depend on the rest part of VOS, this function table is abstraction of those direct calls to VOS/DTX.
  • Most part of this function table is about undo log callbacks, we might want to separate those functions to a dedicated function table for undo log in the future. So both evtree & dbtree can share the same definition of undo log.
