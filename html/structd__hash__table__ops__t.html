◆ hop_rec_decref

bool(* hop_rec_decref(struct d_hash_table *htable, d_list_t *link)

Optional, release refcount on the record link

If this function is provided, it is called while deleting a record from the hash table.

If hop_rec_free() is provided, this function can return true when the refcount reaches zero, in this case, hop_free() will be called. If the record should not be automatically freed by the hash table despite of refcount, then this function should never return true.

Parameters
[in]htablehash table
[in]linkThe link being released.
Return values
falseDo nothing
trueOnly if refcount is zero and the hash item can be freed. If this function can return true, then hop_rec_free() should be defined.

int(* hop_rec_ndecref(struct d_hash_table *htable, d_list_t *link, int count)

Optional, release multiple refcount on the record link

This function expands on hop_rec_decref() so the notes from that function apply here. If hop_rec_decref() is not provided then hop_rec_ndecref() shouldn't be either.

Parameters
[in]htablehash table
[in]linkThe link being released.
[in]countThe number of refcounts to be dropped.
Return values
0Do nothing
1Only if refcount is zero and the hash item can be freed. If this function can return true, then hop_rec_free() should be defined. negative value on error.
